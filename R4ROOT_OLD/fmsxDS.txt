fmsxDS ver.0.09

任天堂DS上で動くMSXエミュレーターのテスト
どうしても試したい人だけどうぞ。

◇免責
  このソフトを使って、あなたの 任天堂DS や メモリカード が壊れたとしても、
  目的が達成できず、時間を無駄にしたとしても、にゃごすは一切の責任を負いません。
  このソフトはSDカードに書き込みを行います。よって内容が壊れる可能性が多分にあります。
  重要なファイルが入っているとは思いませんが、バックアップはこまめにとってください。

◇必須条件
  マジコン等の使用方法を知っていること。
  エミュレーターについてある程度知識があること。
  できない事を、あきらめられること。

◇今回のバージョンアップ
  マウス（とってつけただけ）
  サウンド
  メニュー

◇ファイル
  /
    fmsxDS.nds         ... 実行ファイル
    fmsxDS/
       BIOS/           ... BIOSファイルフォルダ
       ROM/            ... ロムイメージフォルダ
       DSK/            ... ディスクイメージフォルダ
       CAS/            ... カセットイメージフォルダ
       RES/            ... リソースファイルフォルダ
          maru10.bin   ... フォントファイル
       LUA/            ... スクリプトフォルダ
          debug.lua    ... デバッグスクリプト
          keybind.lua  ... キーバインドスクリプト
       fmsxDS.lua      ... メインスクリプト

◇動かし方
  古いマジコン、NINJADS, R4はDLDIドライバインストール済みです。
  それ以外のマジコンの場合は適宜、DLDIドライバのインストールが必要です。
  メモリカードをフォーマットします。
  zipに含まれているフォルダ構成ごとルートにコピーします。
  (スクリプトを書き換えることで変更可能)
  その他、各種BIOSROM MSX2の場合(MSX2.ROM MSX2EXT.ROM DISK.ROM)、
  ロムイメージ、ディスクイメージ,カセットイメージをそれぞれのフォルダにコピー。
  DSに差し込んで、起動します。
  メインメニューが表示されれば起動成功です。

◇キー
  PAD      ... ジョイスティック１上下左右
  A,B      ... ボタン１，２
  START    ... メニュー表示

◇メニューでのキー操作
  PAD      ... 上下
  A        ... 選択
  B        ... キャンセル

◇制限事項
  ・画面の同期を無視していますので、画面が崩れることがあります。
  ・音はPSGとSCCのみです。
  ・ファイル選択画面のタッチでの選択はできません。
  ・ファイル選択画面はまだ２バイト対応されていません。が選択はできるハズです。
